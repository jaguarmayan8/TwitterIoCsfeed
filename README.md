# TwitterIoCsfeed

Automated daily collection of IOCs from public feeds for cybersecurity home labs and SOCs.

## Features

- Daily IOC collection from reliable public sources
- Docker support (easy to run)
- Automatic push to GitHub
- Runs every day at 8:00 AM Central Time
